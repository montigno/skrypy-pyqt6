Troubleshooting

Bugs with Qt

  1. Error with “xcb”:

Solution:

sudo apt-get install -y libxcb-cursor-dev

Bug with Matlab

To check if Matlab is recognized:
- go to the blocks library, drag the ‘start_matlab’ block.
- run (click+R).

If there is no error message, Matlab is recognized correctly.

If the message ‘no module Matlab’ appears, go to chapter How to install dependicies.
